
# Marcenaria Cristo Redentor — Site estático

## Como usar
1. Extraia o arquivo `.zip`.
2. Abra `index.html` no navegador para ver o site.
3. Para publicar, envie a pasta para qualquer hospedagem estática (Netlify, Vercel, GitHub Pages, 000webhost, etc.) ou para a pasta `public_html` do seu provedor.

## Como trocar fotos
Substitua as imagens na pasta `assets/img` mantendo os mesmos nomes de arquivo para atualizar o layout:
- `cozinha.png`, `cama.png`, `banco_jardim.png`, `porta.png`, `logo.png`
